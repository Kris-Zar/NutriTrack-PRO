# NutriTrack Pro

NutriTrack Pro is a full-stack nutrition tracking application with
AI-based image analysis and calorie tracking.

## Tech Stack
- React (Frontend)
- Python (Backend)
- MongoDB
- Google AI APIs

## Features
- Image-based food analysis
- Calorie tracking dashboard
- User profile management

## Local Setup (Required)
This project depends on external services.

### Requirements
- Node.js
- Python
- MongoDB (running locally)
- Google API key (Vision / AI)

### Environment Variables
Create a `.env` file using `.env.example`.

### Known Limitations
- Image analysis will fail without a valid Google API key
- Profile saving requires MongoDB to be running
